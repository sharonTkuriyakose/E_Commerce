An immersive, high-performance e-commerce application built with the MERN stack, featuring interactive 3D product previews and a sleek, modern UI.

---

## ✨ Key Features

- **🎨 Immersive 3D Previews**: Interactive product models using Three.js and React Three Fiber.
- **🔐 Secure Authentication**: JWT-based login and registration with protected routes.
- **🛒 Dynamic Shopping Cart**: Real-time cart management and persistence.
- **📱 Futuristic UI/UX**: Built with Tailwind CSS and Framer Motion for smooth animations and glassmorphism effects.
- **📦 Order Management**: Full checkout flow with order history tracking.

---

## 🛠️ Technology Stack & Uses

### Frontend
- **⚛️ React 18**: Used for building the core user interface with a component-based architecture for scalability and speed.
- **📦 Three.js (R3F)**: Powers the interactive 3D hero section and product previews, allowing users to orbit and inspect gadgets in 3D space.
- **🎬 Framer Motion**: Utilized for luxurious micro-interactions, page transitions, and staggered animations that create a premium feel.
- **🎨 Tailwind CSS**: Provides a utility-first styling system to implement a futuristic dark-mode aesthetic with glassmorphism.
- **🛡️ React Router**: Manages seamless navigation between Home, Products, Cart, and Protected Routes like Checkout.

### Backend
- **🟢 Node.js & Express**: Forms the backbone of the API layer, handling structured requests for products, users, and orders.
- **🍃 MongoDB (Mongoose)**: A NoSQL database used to store product data, user credentials, and order history with high flexibility.
- **🔐 JWT (JSON Web Tokens)**: Ensures secure session management and protects sensitive API endpoints.
- **🔑 Bcryptjs**: Used for industrial-grade password hashing to protect user data privacy.

---

## 🚀 Impact & Purpose
RDX Store is designed to bridge the gap between traditional web shopping and physical product inspection. By leveraging 3D technologies, it increases user engagement and provides a high-end platform for tech enthusiasts to explore premium gadgets before purchasing.
